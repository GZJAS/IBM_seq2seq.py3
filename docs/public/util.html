Util¶

checkpoint¶

class seq2seq.util.checkpoint.Checkpoint(root_dir, model, optimizer_state_dict, epoch, step, input_vocab, output_vocab)¶

The Checkpoint class manages the saving and loading of a model during training. It allows training to be suspended and resumed at a later time (e.g. when running on a cluster using sequential jobs).

To make a checkpoint, initialize a Checkpoint object with the following args; then call that object’s save() method to write parameters to disk.

Parameters:
  • root_dir (str) – main experiment directory
  • model (seq2seq) – seq2seq model being trained
  • optimizer_state_dict (dict) – stores the state of the optimizer
  • epoch (int) – current epoch (an epoch is a loop through the full training data)
  • step (int) – number of examples seen within the current epoch
  • input_vocab (Vocabulary) – vocabulary for the input language
  • output_vocab (Vocabulary) – vocabulary for the output language
Variables:
  • CHECKPOINT_DIR_NAME (str) – name of the checkpoint directory
  • MODEL_DIR_NAME (str) – name of the directory storing models
  • INPUT_VOCAB_FILE (str) – name of the input vocab file
  • OUTPUT_VOCAB_FILE (str) – name of the output vocab file

save()¶

Saves the current model and related training parameters into a subdirectory of the checkpoint directory. The name of the subdirectory is the current local time in Y_M_D_H_M_S format.

Returns:path to the saved checkpoint subdirectory
Return type:str

custom_time¶

seq2seq.util.custom_time.pretty_interval(start_time)¶

Given a previous point in time (e.g. the start of process, measured by calling time.time()), calculates the time elapsed since that point, divides the elapsed time days/hours/minutes/seconds, and returns that summary

Parameters:start_time (float) – the start of the interval of time, obtained by calling time.time()
Returns:elapsed time in ‘##d ##h ##m ##s’ format
Return type:str
seq2seq.util.custom_time.pretty_time(timespan_in_seconds)¶

Given a length of time (measured in seconds), divides that timespan into days/hours/minutes/seconds and returns that summary.

Parameters:timespan_in_seconds (float) – the number of seconds in a span of time
Returns:timespan in ‘##d ##h ##m ##s’ format
Return type:str

Examples:

>>> print seq2seq.util.custom_time.pretty_time(426753)
>>> 4d 22h 32m 33
